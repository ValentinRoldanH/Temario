REPASO 2

semanticas estructurales

Div: etiqueta contenedora que no tiene semantica

Header: cabecera de un sitio web o seccion

Main: Define la seccion principal del documento, solo puede haber una por documento

Footer: pie de pagina de un documento

Nav: representa un menú de navegacion

Article: representa una seccion de autocontenido (que por si sola se explica)

Aside: representa un contenido complementario o secundario

Section: representa una seccion o contenido generico

Address: representa una direccion fisica
